Quick start

  1. Open the Printables tab and pick a sheet.
  2. Use Open to fill it in the browser, or Download to save the HTML file locally.
  3. Print from the sheet’s toolbar, or use the browser print dialog (Ctrl+P) and choose Save as PDF.
